# 实训教学笔记（面向小白）

目标：让完全没接触过的人能运行、理解并动手改造两个练手项目：`水果消消乐游戏`（Pygame）和 `学生信息管理系统`（命令行 CRUD）。

## 项目一：水果消消乐游戏（图形小游戏）

- 位置：`水果消消乐游戏/`
- 主文件：`水果消消乐游戏/main.py`（程序入口与全部逻辑均在此文件）
- 资源：`水果消消乐游戏/static/img/`（背景、按钮、水果图片、特效等）

### 面向小白的说明（逐步理解）

1. 启动流程：
   - 程序入口是 `main()`，它会：初始化 Pygame、创建窗口、调用 `load_all_assets()` 加载图片与字体、创建 `Manager` 对象并进入事件循环（见 `main.py`）。
2. 游戏的“大脑”是 `Manager` 类：
   - `Manager` 存储棋盘数据（`self.grid`）、分数（`self.score`）、当前选中单元、游戏状态等。
3. 鼠标交互：
   - 在主循环里每次 `MOUSEBUTTONDOWN` 都会调用 `Manager.handle_mouse(pos)`。该方法负责：开始游戏、退出、记录选中格和判断交换条件。
4. 交换与消除：
   - `process_swap()`：交换两个格子（位置由用户选中），如果交换后没有形成可消除组合，会把两个格子换回。
   - `eliminate_all()`：核心消除逻辑，负责检测所有可消除组合、把对应格子标记为消除、播放消除动画（`play_clear_animation()`）、填补空格（`drop_fruits()`）并计算得分（`calc_score()`）。
5. 游戏结束判定：
   - 时间到（`GAME_TIME`）或出现“死局”（`is_dead_map()` 找不到任何有效交换）时，会显示结束提示并跳转到分数界面（`go_to_score()`）。

### 初学者可做的几件事（练习）

- 修改 `CELL_SIZE` 或 `WIDTH`/`HEIGHT`，观察棋盘变化。说明：`_cell_coords` 根据这些参数计算每个格子屏幕位置。
- 修改 `SCORE_PER_FRUIT` 中某种水果分值，运行一把并记录分数变化。
- 在 `eliminate_all()` 中打印每次消除的位置用于理解连锁消除顺序。

## 项目二：学生信息管理系统（命令行）

- 位置：`学生信息管理系统/`
- 主要文件：
  - `学生信息管理系统/main.py`（菜单与主循环）
  - `学生信息管理系统/module/component.py`（核心逻辑：读写、增删改查、排序）
  - `学生信息管理系统/module/menu.py`（显示菜单与清屏）
  - 数据文件：`学生信息管理系统/module/students.txt`（每行一个 JSON 对象，持久化存储）

### 面向小白的说明（逐步理解）

1. 启动流程：
   - 运行 `python main.py`，程序会显示菜单并等待用户输入选项。
2. 数据持久化方式：
   - 所有学生信息以 JSON 字符串一行一条写入 `students.txt`。程序通过 `read_students()` 读取所有行并 `json.loads()` 转回 Python 对象列表。
3. 主要功能点（方法）
   - `read_students()` / `save_students(students)`：文件读写接口。若文件不存在，`read_students()` 返回空列表。
   - `input_score(prompt)`：安全输入成绩，会重复提示直到输入 0--100 的整数。
   - `show_students(students)`：格式化打印学生表格（学号、姓名、三科成绩、总分）。
   - `Component` 类下的静态方法：实现添加、查询、删除、修改、排序、统计等操作。

### 初学者可做的几件事（练习）

- 增加“按姓名模糊查询”功能（部分匹配），修改 `Component.search_student()`。
- 在保存前对学生 ID 做更严格的唯一性校验（演示如何防止重复）。
- 将 `students.txt` 内容导出为 CSV 格式（练习文件读写）。

## 教学计划（建议）

- 课前准备（10 分）：确保所有学生安装 Python 与 Pygame，能运行 `python --version` 与 `pip show pygame`。
- 第一节（30 分）：讲解并运行 `学生信息管理系统`，完成一次增删改查练习。
- 第二节（60 分）：讲解 `水果消消乐游戏`：阅读 `main()` 与 `Manager`，演示一次交换→消除→连锁→计分的完整流程。
- 第三节（30 分）：动手改小功能（修改分值、增加模糊查询或打印消除日志），并汇报结果。

## 常见问题与调试提示

- 程序崩溃并提示找不到图片：确认 `水果消消乐游戏/static/img/` 下对应图片存在。
- 运行 `pygame` 出现显示问题（黑屏或窗口卡顿）：检查 Pygame 版本是否兼容，并在代码中避免在主循环中使用长时间阻塞操作。
- 学生数据解析失败（JSON 错误）：检查 `students.txt` 是否被手动破坏或含空行；`read_students()` 已过滤空行。
